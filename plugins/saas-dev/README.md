# saas-dev Plugin — v4.0.0

Enterprise SaaS scaffolding for Django + React/Next.js.

## Install in Claude Code

```bash
/plugin marketplace add Ayithamsetty-Vamsi-krishna/claude-skills
/plugin install saas-dev@vamsi-claude-skills
```

## What's included

107 reference files across 9 skills:

| Skill | Covers |
|---|---|
| `using-saas-dev` | Bootstrap — auto-wires the pipeline at session start |
| `saas-dev-brainstorm` | Socratic design before writing code |
| `saas-dev-plan` | Breaks spec into 2–5 min tasks |
| `saas-dev-execute` | Subagent per task + two-stage review |
| `django-project-setup` | New project + CLAUDE.md v2 |
| `django-backend-dev` | DRF + audit + multi-tenancy + search + encryption + GDPR |
| `django-auth-dev` | 2FA + JWT + RBAC |
| `django-integrations-dev` | Payments + PDF + webhooks |
| `django-devops-dev` | Logging + metrics + tracing + K8s |
